estructura selectiva doble
En un juego de preguntas a las que se responde SI o NO gana quien responda correctamente las tres preguntas. Si se responde mal a cualquiera de ellas ya no se pregunta la siguiente y termina el juego. Las preguntas son: ¿Colón descubrió América? ¿El lago Titicaca es el lago navegable más alto del mundo? ¿El español es el único idioma oficial de Perú?
Determinar si un alumno aprueba o reprueba un curso, sabiendo que aprobara si su promedio de tres calificaciones es mayor o igual a 70; reprueba en caso contrario.

#include<iostream>
using namespace std;
int main(){
int cal1,cal2,cal3,pro;
cout<<"escribir la primera calificacion"<<endl;
cin>>cal1;
cout<<"escribir la segunda calificacion"<<endl;
cin>>cal2;
cout<<"escribir la tercera calificacion"<<endl;
cin>>cal3;
pro=(cal1+cal2+cal3)/3;
if(pro>=70){
	cout<<"esta aprobado";
}else{
	cout<<"esta desaprobado";
}
return 0;
}
